Учебный репозиторий для практики работы с HTTP-сервером на Go 
Содержит два проекта с обработкой HTTP-запросов и потокобезопасной работой с данными

---

## Проекты

### 1. Payments Example (payment_with_JSON)
Простой сервер для обработки платежей

- Эндпоинт: `/pay` — принимает JSON с платежом  
- Структура запроса:

{
  "description": "Покупка книги",
  "usd": 50,
  "fullName": "Иван Иванов",
  "addres": "Москва, ул. Пушкина, 10"
}

- Проверяет баланс и списывает сумму покупки 
- Хранит историю платежей в `paymentHistory` 
- Потокобезопасное изменение данных с помощью `sync.Mutex`

---

### 2. Event Logger (event_manager)
HTTP-сервер для работы с событиями

- Эндпоинты:  
  - `/event/add` — добавить событие (JSON с `type` и `message`)  
  - `/event/list` — получить список всех событий  
  - `/event/typeFilter?type=TYPE` — получить события конкретного типа  
  - `/event/clear` — очистить все события  

- Потокобезопасное хранение событий (`sync.Mutex` + `atomic.Int64` для ID).  
- Использует JSON для передачи данных и хранения времени события 

---

## Используемые технологии

- Go (Golang)  
- HTTP-сервер (`net/http`)  
- JSON (`encoding/json`)  
- Потокобезопасность: `sync.Mutex`, `sync/atomic`  
- Работа с запросами и телом HTTP (`http.Request.Body`)  
- Обработка ошибок и статус-коды HTTP
